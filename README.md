# 🎰 Однорукий бандит (Slot Machine)

Курсовая работа по дисциплине "Основы Программирования"
Игра "Однорукий бандит" реализована на языке C# с использованием Windows Forms.

---

## 📋 Описание

"Однорукий бандит" — это игра на основе игрового автомата.
Игрок делает ставку и крутит барабаны. При совпадении символов
игрок получает выигрыш. Цель игры — увеличить свой баланс.

---

## 🎮 Правила игры

- Начальный баланс игрока: **100 монет**
- Игрок выбирает размер ставки (от 1 до 100)
- После нажатия кнопки **КРУТИТЬ** барабаны случайно прокручиваются
- Результат зависит от выпавших символов:

| Результат | Выигрыш |
|-----------|---------|
| Три одинаковых символа | Ставка × 5 (ДЖЕКПОТ) |
| Два одинаковых символа | Ставка × 2 |
| Нет совпадений | Ставка сгорает |

- Если баланс достигает 0 — игра заканчивается

## 🏗️ Структура проекта

Проект состоит из следующих основных файлов:

- `Program.cs` — содержит точку входа в приложение.
- `MainForm.cs` — реализует взаимодействие пользователя с интерфейсом.
- `MainForm.Designer.cs` — содержит автоматически созданный код элементов формы.
- `MainForm.resx` — файл ресурсов формы.
- `SlotMachine.cs` — реализует основную игровую логику.
- `Player.cs` — описывает игрока, его баланс и ставку.
- `Symbols.cs` — содержит набор символов игрового автомата.
- `WinChecker.cs` — проверяет выпавшую комбинацию и вычисляет выигрыш.
- `SlotMachine.csproj` — файл конфигурации проекта.
- `README.md` — описание проекта и инструкция по запуску.

## 📦 Классы и их описание

### `Player.cs`
Хранит данные об игроке.
- `Balance` — текущий баланс
- `Bet` — текущая ставка
- `PlaceBet()` — снимает ставку с баланса
- `AddWin()` — добавляет выигрыш к балансу
- `IsBroke()` — проверяет, закончились ли деньги

### `Symbols.cs`
Хранит список символов барабанов.
- `GetSymbols()` — возвращает массив символов
- `Count()` — возвращает количество символов

### `WinChecker.cs`
Проверяет результат и считает выигрыш.
- `Check()` — считает сумму выигрыша
- `GetMessage()` — возвращает текст результата

### `SlotMachine.cs`
Главный класс игры. Связывает все классы вместе.
- `Spin()` — запускает вращение барабанов
- `GetResultMessage()` — возвращает результат хода

### `MainForm.cs`
Отвечает только за интерфейс.
- Отображает символы, баланс, сообщения
- Обрабатывает нажатия кнопок
